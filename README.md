# Mi Primera App en Angular - UTN BA Centro de e-Learning

## Descripción del Proyecto
Este proyecto corresponde a la actividad práctica del Módulo 1 - Unidad 1 del curso "Conociendo Angular". Su objetivo principal es comprender el flujo básico de trabajo con el framework, la utilización de los comandos esenciales de Angular CLI, el reconocimiento de la estructura interna del proyecto y la implementación de la interpolación de datos en la vista.

---

## Instrucciones de Instalación y Ejecución

Para clonar, instalar las dependencias y ejecutar este proyecto de forma local, siga estos pasos en su terminal:

### 1. Clonar el repositorio
Descargue una copia local del proyecto ejecutando:
```bash
git clone git@github.com:mazypincha/mi-primera-app.git
```

### 2. Instalar dependencias
```bash
cd mi-primera-app
npm install
```

### 3. Ejecutar la aplicación
```bash
ng serve
```

### 4. Acceso en el navegador
http://localhost:4200/
